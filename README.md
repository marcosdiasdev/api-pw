# Produtos API

API para controle de produtos e fabricantes, desenvolvida com Node.js, Express, Knex e SQLite.

## Rotas

- `GET /produtos`
- `GET /produtos/:id`
- `POST /produtos`
- `PATCH /produtos`
- `DELETE /produtos`
- `GET /fabricantes`

## Acesso

Acesse a API em [https://api-pw-2021-1.herokuapp.com/produtos](https://api-pw-2021-1.herokuapp.com/produtos).
